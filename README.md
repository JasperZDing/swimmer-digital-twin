# 🏊 Swimmer Digital Twin — Project Overview

## Goal
Build a digital twin system for a swimmer — simulating or analyzing motion, performance, race strategy, or training feedback. Targeted at one of Yongwang's kids.

## Scope Strategy
**Think big, start small.** Start with the simplest version that still demonstrates the core idea.

## Potential Features (brainstorm)
- [ ] Stroke detection / count
- [ ] Lap counting via sensor or video
- [ ] Breath timing analysis
- [ ] Turn efficiency scoring
- [ ] Race simulation / playback
- [ ] Training load tracking
- [ ] 3D motion visualization
- [ ] Real-time race commentary

## Tech Directions
- **Hardware:** Phone camera, wearable sensors (IMU), pressure pads
- **Simulation:** Physics-based stroke model, biomechanics engine
- **Visualization:** 3D swimmer model (Three.js, Unity, Unreal)
- **Data:** Time series from sensors, video processing with pose detection
